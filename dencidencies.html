Diagnóstico interno: Gestión de incidencias técnicas

  • Las incidencias se comunican de forma verbal o en papel.
  • No existe un sistema centralizado para el seguimiento.
  • El equipo técnico puede no recibir el aviso de forma inmediata.
  • No se recopilan datos para analizar problemas recurrentes o carga de trabajo.

Propuestas de soluciones digitales

  • Implementar una plataforma online para registrar incidencias.
  • Usar un sistema con estados (pendiente, en curso, resuelto).
  • Enviar notificaciones automáticas al equipo técnico.
  • Generar informes y estadísticas automáticas del servicio técnico.

Objetivos del PTD

  • Organizar y centralizar la gestión técnica.
  • Hacer un seguimiento claro y transparente de cada incidencia.
  • Agilizar los tiempos de respuesta y mejorar la eficiencia técnica.
  • Analizar los datos para optimizar recursos y servicios.
